# Telegram Link Analyzer Bot

Telegram으로 전달한 링크를 읽고 한국어 요약, 인사이트, 관련 종목을 짧은 개조식으로 돌려주는 FastAPI 기반 봇입니다.

## Current Output Format

봇 응답은 한 메시지 안에 들어오도록 아래 형식으로 압축됩니다.

```text
[핵심 요약]
- bullet 1개

[인사이트]
- bullet 5개

[관련 종목]
- 종목명: 영향 방향, 투자 체크포인트
- 종목명: 영향 방향, 투자 체크포인트
- 종목명: 영향 방향, 투자 체크포인트
```

## Today's Improvements

- 전문 번역 제거, 한국어 개조식 요약 중심으로 단순화
- `[핵심 요약]`을 bullet 1개로 축소해 메시지 길이 단축
- `[인사이트]`를 5개 bullet로 확장하고 반대 시나리오와 기사 신뢰 가능성까지 함께 검토
- `[관련 종목]` 섹션 추가
- 관련 종목은 코스피·코스닥 종목 3개만 고르고, 각 종목별 영향 방향과 주식 투자 체크포인트를 한 줄로 제시
- 전체 응답이 개조식 형식에서 벗어나지 않도록 후처리 보정 추가

## Notes

- `[관련 종목]`은 투자 참고용 분석이며, 직접적인 매수·매도 권유는 하지 않습니다.
- 동적 사이트 대응을 위해 Playwright 기반 추출을 사용합니다.
- arXiv 링크는 PDF 본문 우선, 실패 시 초록 폴백으로 처리합니다.
